# Open_Annotation

![](demo_openannotation.gif)

This software is for bounding box annotation. It can be in collaboration with [iPhotoDraw](http://iphotodraw-free-software.fileflash.com).

## Step 1

Use iPhotoDraw to get all the bounding box you want.

## Step 2

Use BBOX_JXU to annotate those bounding box, detail user guide can be found in "doc".


If you have any question, please contact me.
